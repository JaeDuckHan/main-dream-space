회원 관리 화면 구성 (공통)

테이블: 아바타 | 이름/이메일 | 가입일 | 소셜로그인 | 역할 | 상태 | 주문수
검색: 이름·이메일 검색 + 역할 필터 + 상태 필터
클릭 → 사이드 패널:
  • 기본 정보
  • 역할 변경 (user ↔ admin)
  • 계정 비활성화
  • 작성 글 목록 탭
  • 업체 등록 현황 탭
  • 주문 내역 탭 (서브시스템 3 연동)
